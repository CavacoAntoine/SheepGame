Ceci est le README du jeu "sheep game" par Arthur Delignette et Antoine Cavaco.

Pour pouvoir jouer au jeu, il faut placer toutes les images contenues dans le dossier média,
à la racine du projet à côté de SheepGame.exe.

Ensuite pour le lancer il suffit d'executer la ligne de commande : 
.\SheepGame.exe nombre_de_moutons nombre_de_loups temps_de_jeu

Le but du jeu est de garder en vie le maximum de mouton. Mais attention aux loups !
Vous pouvez les faire fuire en vous déplacant avec z,q,s,d. Ainsi votre chien pourra faire fuire les loups.
Mais ce n'est pas fini !
Les moutons morts reviendront à la vie pour tout dévorer !
Survivez à cette deuxième phase en frappant les moutons zombies les plus proches !

A noter que les moutons zombies mangent tout le monde et tout les animaux morts se transforment en mouton zombie.
De plus, les moutons normaux vont essayer de s'accoupler.
